# Screenshots Directory

This directory contains screenshots of the Regency Horror Menu at various resolutions and states.

## Screenshot List

1. **desktop-menu.png** - Main menu view on desktop (1920x1080)
2. **desktop-editor.png** - Script editor view on desktop (1920x1080)
3. **mobile-16-9.png** - Menu on 16:9 mobile device (375x667)
4. **mobile-20-9.png** - Menu on 20:9 mobile device (360x800)
5. **hover-state.png** - Menu strip with hover effect
6. **registry-modal.png** - Secret registry modal overlay

## How to Generate Screenshots

Visit the live site at https://anacondy.github.io/3-Regency-Horror-Menu/ and use browser dev tools to:

1. Open Developer Tools (F12)
2. Toggle device emulation (Ctrl+Shift+M)
3. Select desired resolution
4. Take screenshot using browser's screenshot tool

### Recommended Resolutions for Testing

**16:9 Devices:**
- 375x667 (iPhone SE)
- 414x736 (iPhone 8 Plus)
- 360x640 (Galaxy S5)

**20:9 Devices:**
- 360x800 (Galaxy S21)
- 393x851 (Pixel 5)
- 412x915 (OnePlus 9)
